1. 颜色缓冲区
2. 深度缓冲区
3. 模板缓冲区

执行操作的API

```JavaScript
gl.clear(buffer);

/*
 * buffer参数可以选择下面三种之一
 * gl.COLOR_BUFFER_BIT
 * gl.DEPTH_BUFFER_BIT
 * gl.STENCIL_BUFFER_BIT
 */
```
